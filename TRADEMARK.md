# Trademark and Brand Policy

This policy governs trademark and brand usage for OCR Arcade.

## Ownership

- "OCR Arcade" name, logos, and visual branding are property of WinterDrive.
- Domain and official distribution channels are reserved by the project owner.

## What Is Allowed

You may:

- Reference OCR Arcade in factual, nominative ways (for example: "compatible with OCR Arcade").
- Link to the official repository and documentation.

## What Is Not Allowed Without Written Permission

You may not:

- Use "OCR Arcade" name or logo in a way that implies endorsement or official affiliation.
- Rebrand a fork as "OCR Arcade" or as an official release.
- Use confusingly similar names, logos, or product presentation in commercial offerings.

## Fork Naming Guidance

If you fork the project, you should:

- Use a distinct project name and identity.
- Clearly state it is an unofficial fork.
- Remove or replace official logos and brand assets where appropriate.

## Reporting Misuse

Please report potential misuse through:

- GitHub Issues: <https://github.com/winterdrive/OCR-Arcade/issues>
- GitHub Discussions: <https://github.com/winterdrive/OCR-Arcade/discussions>

Open-source code licenses (including AGPL and any commercial software license)
do not grant rights to use project trademarks.
